# 🎸 Chordia

**Chordia** is a web application that allows you to create and visualize chords interactively. Designed for musicians, Chordia provides an easy and intuitive way to create different note combinations.

## 🚀 Technologies Used

- **HTML** – For the application's structure.  
- **JavaScript** – For chord selection and visualization logic.  
- **Tailwind CSS** – For a modern and responsive design.  

## 📌 Features

✔️ Create custom chords.  
✔️ Dynamic visualization of selected chords.  
✔️ Intuitive and user-friendly interface.  
✔️ Responsive design with Tailwind CSS.  

## 📂 Installation & Usage

1. Clone this repository:  
   ```bash
   git clone https://github.com/RubenMRDev/Chordia.git
   ```
2. Navigate to the project folder:  
   ```bash
   cd Chordia
   ```
3. Open `index.html` in your favorite browser.
